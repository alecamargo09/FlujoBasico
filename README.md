# Práctica 2 
# 1. ¿Qué es una Base De Datos Relacional?
Es un sistema donde la información se guarda en tablas organizadas y que pueden conectarse entre sí para mantener los datos ordenados.

# 2. ¿Qué es una Tabla en el contexto Base De Datos Relacionales?
Es el lugar donde se guardan los datos en forma de filas y columnas. Cada tabla representa algo específico, por ejemplo estudiantes o productos.

# 3. ¿Qué es una relación en el contexto de Base De Datos Relacionales?
Es el vínculo que existe entre dos tablas, el cual permite conectar información usando claves.

# 4. ¿Qué representa una columna en el contexto de Base De Datos Relacionales?
Representa una característica del dato, como nombre, edad, dirección o precio.

# 5. ¿Qué representa un renglón en el contexto de Base De Datos Relacionales?
Es un registro completo dentro de la tabla, o sea, todos los datos de un elemento.

# 6. ¿Qué es un motor de base de datos relacional?
Es el programa que se encarga de crear, organizar y administrar la base de datos.

# 7. ¿Cuáles son los motores de bases de datos relacionales mas populares?
MySQL, PostgreSQL, Oracle, SQL Server y MariaDB.

# 8. ¿Cuál es el motor de base datos ligero que no necesita un servidor?
SQLite.

# 9. ¿Qué significa SQL en el contexto de base datos relacionales?
Significa Structured Query Language, que es el lenguaje que se usa para manejar bases de datos.

# 10. ¿Cuáles son los elementos de un sistema de información basado en base de datos relacionales?
La base de datos, el motor de base de datos, los programas que la usan, los usuarios, el hardware y las reglas o procedimientos.
